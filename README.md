## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/ishikajain0203/portfolio_project.git
```

### 2. Go to project folder

```bash
cd portfolio_project
```

### 3. Install dependencies

```bash
cd frontend
npm install
```

### 4. Run the project

```bash
npm start
```
